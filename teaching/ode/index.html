Ordinary Differential Equations

Class Description

Credit: 4 Credits (3-1-0)

Instructor: K. Sreenadh

Course Contents:

Theory of First order Equations: Picard's existence and uniqueness theorems, Peono's existence theorem, singular solutions

Linear system of equations: Linear system of first order equations, Matrix methods, Jordan forms and applications

Series methods: Power series methods, important equations of Mathematics physics, Cauchy-Kowalevski theorem

Stability theory: Linear stabiity, Lyapunov methods

Boundary Value problems: Green's functions and eigenvalue problems and applications to Fourier methods.
